# Setup

Keplr is a popular browser extension wallet for the Cosmos ecosystem. Keplr
features account management, token transfers, transaction signing and more.

To install Keplr, visit the [Keplr website](https://www.keplr.app/) and follow
the instructions.

Once installed, follow the setup procedure to create a new account. You can
either create a new account, import an existing one or use a hardware wallet.

![Keplr](/img/docs/keplr-setup.png)

To add ZetaChain to Keplr:

- go to https://chains.keplr.app/
- Type "ZetaChain" in the search box
- Click "Add to Keplr"

![Keplr](/img/docs/keplr-add-zetachain.png)

Now open the Keplr extension and you should see your ZetaChain balance:
