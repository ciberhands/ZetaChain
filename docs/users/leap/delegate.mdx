# Delegating to a Validator

import LeapStake from "/img/docs/leap-stake.png";
import LeapValidator from "/img/docs/leap-validator.png";
import LeapReview from "/img/docs/leap-review.png";
import LeapConfirm from "/img/docs/leap-confirm.png";

To delegate your ZETA tokens to a validator, select the "Stake" tag in Leap:

<img src={LeapStake} style={{ width: "400px" }} />

Choose a validator to delegate to:

<img src={LeapValidator} style={{ width: "400px" }} />

Enter the amount of ZETA you would like to delegate and click "Review":

<img src={LeapReview} style={{ width: "400px" }} />

Review the transaction details and click "Confirm" to sign and broadcast the
transaction:

<img src={LeapConfirm} style={{ width: "400px" }} />
