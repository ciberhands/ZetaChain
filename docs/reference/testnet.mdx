# ZetaChain Testnet

## Athens 3 Testnet Network Info

Athens 3 is the latest ZetaChain testnet. It is live with ZetaChain's latest
features, but will be undergoing active upgrades and development. Periodically,
this network may undergo stress testing affecting the network's availability.
This network is poised to eventually become the primary testnet for all
ZetaChain development. If you are deploying and testing dApps moving forward,
you should deploy and test them here.

:::tip

For a list of RPC endpoints, refer to the [API endpoints page](/reference/api).

:::
